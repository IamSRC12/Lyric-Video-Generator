
# 🚀 QUICK START - Ultimate Lyric Video Generator

## ⚡ Get Started in 60 Seconds!

### 1. Open the App
```bash
# Double-click this file:
START.bat

# OR run manually:
cd "a:\Antigravity\Project Universe\Lyric-Video-Generator"
start index.html
```

### 2. Create Your First Video

#### **EASY MODE** (AI Does Everything):
1. Upload your audio file
2. Paste lyrics (or leave empty for AI detection)
3. Click "🤖 AI Sync & Analyze"
4. Click "✨ Apply AI Suggestions"
5. Click "🎬 Export Video"
**Done!** ✅

#### **PRO MODE** (Full Control):
1. Upload audio + paste lyrics
2. Click "🤖 AI Sync & Analyze"
3. **NEW!** Open Timeline Editor
4. Drag segments to adjust timing
5. Double-click to edit text
6. Choose from 30+ animations
7. Customize colors and effects
8. Preview and export
**Professional Result!** 🎬

## ⌨️ Timeline Editor Controls

### Mouse:
- **Click**: Select segment
- **Double-Click**: Edit text
- **Drag**: Move segment
- **Drag Handles**: Resize

### Buttons:
- **▶️**: Play/Pause
- **🔍+/-**: Zoom
- **➕**: Add segment
- **✂️**: Split
- **🗑️**: Delete
- **🤖**: AI Re-Sync

→ Choose from 30+ in dropdown

### Colors don't match?
→ Use "✨ Apply AI Suggestions"

### Need to edit text?
→ Double-click segment in timeline
