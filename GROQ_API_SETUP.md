## 🚀 **How to Use**

### **Step 1: Get FREE Groq API Key**

1. Go to: **https://console.groq.com/keys**
2. Sign up (FREE - no credit card!)
3. Click "Create API Key"
4. Copy the key (starts with `gsk_...`)

### **Step 2: Paste API Key**

1. Open the Lyric Video Generator
2. Find "🤖 API Provider" section
3. Select "Groq API (Fast & Free!) 🚀"
4. Paste your API key in the text box
5. Done!

### **Step 3: Use the App**

1. Upload audio
2. Paste lyrics
3. Click "🤖 AI Sync & Analyze"
4. **IT WORKS!** No more "failed to fetch" errors!

## 🆘 **Troubleshooting**

### **"Invalid API Key"**:
- Make sure key starts with `gsk_`
- Copy the entire key
- No extra spaces
- Try regenerating key

### **"Rate Limit"**:
- Wait 1 minute
- Try again
- Groq allows 20/minute (plenty!)

### **Still Not Working?**:
1. Clear browser cache (Ctrl+Shift+R)
2. Check API key is pasted correctly
3. Make sure "Groq API" is selected
