## 🚀 **Quick Start**

1.  **Double-click `START.bat`**
2.  Select your AI Model Quality (1-4)
    -   *Recommended: Option 2 (Small) or 3 (Medium)*
3.  Wait for the AI to load (First time takes longer to download models).
4.  The app will open automatically!

## 🛠️ **Troubleshooting**

- **"Docker not running"**: Ideally, open Docker Desktop before running `START.bat`.
- **Slow First Run**: The AI needs to download the model (approx 1-3GB depending on choice). Be patient!
